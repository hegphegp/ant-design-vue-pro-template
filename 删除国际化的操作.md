### 所有的catch都要打印错误，即使往外抛错误也要打印
* 所有的catch都要打印错误，即使往外抛错误也要打印
```
# 后端接口返回的数据错误时，必须打印日志，没打印，就是严重失误
# console打印错误日志的正确方式，必须全红，必须粗体，必须大字体， %c 开头表示配置样式
catch(error => {
    console.error('%c ' + errMsg, 'font-weight:bold; font-size:13px;')
    reject(error)
})
```
